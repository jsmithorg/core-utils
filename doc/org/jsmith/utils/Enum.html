Packageorg.jsmith.utils
Classpublic class Enum

Base class for creating strongly-typed enumerations.

The class should not be used by itself - it should only be subclassed.


Example
To create a strongly typed enumeration, create a class that subclasses Enum 
    
    public class MyEnum extends Enum
    {
     ...
    }
Inside the class, create constants using the inherited create method: 
  
    public class MyEnum extends Enum
    {
     public static const MY_CONST_1:MyEnum = create("myConst1", MyEnum);
     public static const MY_CONST_2:MyEnum = create(2, MyEnum);
     public static const MY_CONST_3:MyEnum = create(MyObject, MyEnum);
    }
